# Getting Started

## What Shields.io Is

- Shields.io provides concise, consistent, legible badges for GitHub READMEs and other web pages.
- The service supports badges for many CI services, package registries, distributions, app stores, social networks, code coverage services, and code analysis services.

## Basic Workflow

- Browse the badge catalog or use search to find a badge.
- Use the builder to fill in required path parameters such as user, repo, package, or slug.
- Optionally customize the badge with label, colors, logo, and style settings.
- Copy the badge URL or code snippet into a README or web page.

## Main Entry Points

- Dynamic badges
- Static badges
- Badge-Maker NPM library
- Host your own instance
