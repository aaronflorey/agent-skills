# fish\_default\_key\_bindings - set emacs key bindings for fish[¶](#fish-default-key-bindings-set-emacs-key-bindings-for-fish "Link to this heading")

## Synopsis[¶](#synopsis "Link to this heading")

fish\_default\_key\_bindings
COPY

## Description[¶](#description "Link to this heading")

`fish_default_key_bindings` sets the emacs key bindings for `fish` shell.

Some of the Emacs key bindings are defined [here](../interactive.html#emacs-mode).

There are no parameters for `fish_default_key_bindings`.
