# dirs - print directory stack[¶](#dirs-print-directory-stack "Link to this heading")

## Synopsis[¶](#synopsis "Link to this heading")

dirs \[\-c\]
COPY

## Description[¶](#description "Link to this heading")

`dirs` prints the current [directory stack](../interactive.html#directory-stack), as created by [pushd](pushd.html) and modified by [popd](popd.html).

The following options are available:

**\-c**:

Clear the directory stack instead of printing it.

**\-h** or **\--help**

Displays help about using this command.

`dirs` does not accept any arguments.
