# Responses

Source page: https://docs.saloon.dev/the-basics/responses

Common response helpers include:
- `json()`
- `dto()`
- `dtoOrFail()`
- `status()`
- `successful()`
- `failed()`
- `throw()`
- `getPsrRequest()`
- `getPsrResponse()`

- Use custom response classes when a package needs richer domain behavior than the default `Response` wrapper.
