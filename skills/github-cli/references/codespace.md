# gh codespace

- Docs: `https://cli.github.com/manual/gh_codespace`
- Use to create, inspect, SSH into, stop, rebuild, or delete Codespaces
- Key subcommands: `list`, `create`, `ssh`, `code`, `ports`, `logs`, `stop`, `delete`, `view`
- Example:

```bash
gh codespace list
gh codespace ssh -c my-codespace
gh codespace stop -c my-codespace
```
