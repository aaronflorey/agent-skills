# gh label

- Docs: `https://cli.github.com/manual/gh_label`
- Use to create, edit, list, clone, or delete labels
- Key subcommands: `list`, `create`, `edit`, `clone`, `delete`
- Common flag: `-R OWNER/REPO`
- Example:

```bash
gh label list
gh label create bug --color ff0000 --description "Something is broken"
```
