# Documentation Generation

Cobra can automatically generate documentation for your CLI in multiple formats.

## Setup

```go
import "github.com/spf13/cobra/doc"
```

## Markdown Documentation

### Generate Full Tree

```go
err := doc.GenMarkdownTree(rootCmd, "./docs")
if err != nil {
    log.Fatal(err)
}
```

This creates one `.md` file per command in the `./docs` directory.

### Generate Single Command

```go
buf := new(bytes.Buffer)
err := doc.GenMarkdown(cmd, buf)
if err != nil {
    log.Fatal(err)
}
fmt.Println(buf.String())
```

### Custom Markdown

```go
// Custom link handler and file prepender
doc.GenMarkdownTreeCustom(rootCmd, "./docs", filePrepender, linkHandler)

// filePrepender adds content before each file (e.g., Hugo front matter)
filePrepender := func(filename string) string {
    now := time.Now().Format(time.RFC3339)
    name := filepath.Base(filename)
    base := strings.TrimSuffix(name, path.Ext(name))
    url := "/commands/" + strings.ToLower(base) + "/"
    return fmt.Sprintf(`---
date: %s
title: "%s"
slug: %s
url: %s
---
`, now, strings.Replace(base, "_", " ", -1), base, url)
}

// linkHandler customizes internal links
linkHandler := func(name string) string {
    base := strings.TrimSuffix(name, path.Ext(name))
    return "/commands/" + strings.ToLower(base) + "/"
}
```

## Man Pages

### Generate Full Tree

```go
header := &doc.GenManHeader{
    Title:   "MYAPP",
    Section: "1",
}
err := doc.GenManTree(rootCmd, header, "./man")
if err != nil {
    log.Fatal(err)
}
```

### GenManHeader Fields

| Field | Description | Example |
|-------|-------------|---------|
| `Title` | Man page title (uppercase) | `"MYAPP"` |
| `Section` | Man section number | `"1"` (user commands) |
| `Date` | Date string | `"2024-01-15"` |
| `Source` | Source/version | `"MyApp 1.0"` |
| `Manual` | Manual name | `"MyApp Manual"` |

### Man Page Sections

| Section | Content |
|---------|---------|
| 1 | User commands |
| 2 | System calls |
| 3 | Library functions |
| 5 | File formats |
| 7 | Miscellaneous |
| 8 | System admin commands |

## Common Options

### Disable Auto-Generated Tag

By default, docs include "Auto generated by spf13/cobra...". To remove:

```go
cmd.DisableAutoGenTag = true

// Or for the entire tree
func disableAutoGen(cmd *cobra.Command) {
    cmd.DisableAutoGenTag = true
    for _, c := range cmd.Commands() {
        disableAutoGen(c)
    }
}
disableAutoGen(rootCmd)
```

### Document Completion Command

The default completion command isn't documented by default. To include it:

```go
rootCmd.InitDefaultCompletionCmd()
```

## Build-Time Documentation

Common pattern for generating docs at build time:

```go
// cmd/gendocs.go (use build tag to exclude from main binary)
//go:build ignore

package main

import (
    "log"
    "myapp/cmd"
    "github.com/spf13/cobra/doc"
)

func main() {
    err := doc.GenMarkdownTree(cmd.RootCmd, "./docs")
    if err != nil {
        log.Fatal(err)
    }
}
```

Run with:
```bash
go run cmd/gendocs.go
```

Or add to Makefile:
```makefile
docs:
	go run cmd/gendocs.go
```

## Example: Complete Documentation Generator

```go
package main

import (
    "log"
    "os"
    "path/filepath"
    "strings"
    "time"

    "myapp/cmd"
    "github.com/spf13/cobra/doc"
)

func main() {
    // Disable auto-gen tag
    disableAutoGen(cmd.RootCmd)
    
    // Generate markdown
    mdDir := "./docs/commands"
    os.MkdirAll(mdDir, 0755)
    
    filePrepender := func(filename string) string {
        name := filepath.Base(filename)
        base := strings.TrimSuffix(name, filepath.Ext(name))
        return fmt.Sprintf(`---
title: %s
description: %s command reference
---

`, base, base)
    }
    
    linkHandler := func(name string) string {
        base := strings.TrimSuffix(name, filepath.Ext(name))
        return "./" + strings.ToLower(base) + ".md"
    }
    
    if err := doc.GenMarkdownTreeCustom(cmd.RootCmd, mdDir, filePrepender, linkHandler); err != nil {
        log.Fatal(err)
    }
    
    // Generate man pages
    manDir := "./man/man1"
    os.MkdirAll(manDir, 0755)
    
    header := &doc.GenManHeader{
        Title:   "MYAPP",
        Section: "1",
        Source:  "MyApp " + cmd.Version,
    }
    
    if err := doc.GenManTree(cmd.RootCmd, header, manDir); err != nil {
        log.Fatal(err)
    }
    
    log.Println("Documentation generated successfully")
}

func disableAutoGen(cmd *cobra.Command) {
    cmd.DisableAutoGenTag = true
    for _, c := range cmd.Commands() {
        disableAutoGen(c)
    }
}
```

## CI Integration

### GitHub Actions Example

```yaml
name: Generate Docs

on:
  push:
    branches: [main]
    paths:
      - 'cmd/**'

jobs:
  docs:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - uses: actions/setup-go@v4
        with:
          go-version: '1.21'
      
      - name: Generate docs
        run: go run ./scripts/gendocs.go
      
      - name: Commit docs
        run: |
          git config user.name "GitHub Actions"
          git config user.email "actions@github.com"
          git add docs/
          git diff --staged --quiet || git commit -m "docs: update command reference"
          git push
```
