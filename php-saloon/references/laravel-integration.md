# Laravel Integration

Source page: https://docs.saloon.dev/installable-plugins/laravel-integration

- Install the Laravel plugin as an optional dependency when packaging Laravel support.
- The docs include Artisan generators for connectors, requests, responses, plugins, and authenticators.
- The Laravel facade supports mocking helpers.
- Package Laravel support in its own namespace, with config, bindings, and a facade.
- Use `spatie/laravel-package-tools` for package wiring around config and service-provider setup.
