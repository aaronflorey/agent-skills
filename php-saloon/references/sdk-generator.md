# SDK Generator

Source page: https://docs.saloon.dev/installable-plugins/sdk-generator

- The SDK generator is a third-party tool for generating Saloon foundations from OpenAPI files or Postman collections.
- It can accelerate the first pass of connectors, requests, and responses.
- Treat generated output as a starting point, not a finished SDK.
- Reapply the rules from `standalone-sdk-guidance.md` after generation.
