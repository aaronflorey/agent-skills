# gh auth

- Docs: `https://cli.github.com/manual/gh_auth`
- Use for login, token refresh, host/account switching, and git credential setup
- Key subcommands: `login`, `status`, `switch`, `refresh`, `setup-git`, `token`, `logout`
- Example:

```bash
gh auth login
gh auth status
gh auth refresh -s project
gh auth token
```
