# gh gist

- Docs: `https://cli.github.com/manual/gh_gist`
- Use to create, clone, edit, rename, view, list, or delete gists
- Key subcommands: `create`, `clone`, `edit`, `rename`, `view`, `list`, `delete`
- Example:

```bash
gh gist create notes.txt -d "Quick notes"
gh gist list
gh gist view <gist-id>
```
