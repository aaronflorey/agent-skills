# gh browse

- Docs: `https://cli.github.com/manual/gh_browse`
- Use to open repo pages, files, issues, PRs, releases, settings, or actions in a browser
- Useful flags: `--no-browser`, `--branch`, `--commit`, `--blame`, `--settings`, `--releases`, `-R`
- Example:

```bash
gh browse
gh browse 123
gh browse src/main.go:42 --no-browser
```
