# gh run

- Docs: `https://cli.github.com/manual/gh_run`
- Use to inspect workflow runs, download artifacts, rerun, cancel, delete, or watch runs
- Key subcommands: `list`, `view`, `watch`, `download`, `rerun`, `cancel`, `delete`
- Common flag: `-R OWNER/REPO`
- Example:

```bash
gh run list
gh run view <run-id>
gh run watch <run-id>
```
